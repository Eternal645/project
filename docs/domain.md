# Предметная область

## Сущности

- `Товары`: мебельные позиции, цена, остаток и место хранения.
- `Категории`: группировка товаров.
- `Клиенты`: покупатели и контактные данные.
- `Сотрудники`: менеджеры и курьеры.
- `Заказы`: покупка клиента, сумма, дата, статус и способ оплаты.
- `СтрокиЗаказа`: товары внутри заказа.
- `Поставщики`: контрагенты для поставок.
- `Поставки`: поступление товаров от поставщиков.
- `Доставки`: доставка заказа клиенту.
- `СтатусыЗаказа`, `СтатусыДоставки`, `СпособыОплаты`, `МестаХранения`: справочники.

## Правила

- Количество товара в строке заказа должно быть больше нуля.
- Цена товара и цена строки заказа не могут быть отрицательными.
- Остаток товара не может быть отрицательным.
- При переходе заказа в статус с флагом `is_delivering` товары списываются со склада.
- При отмене заказа, который уже был в доставке, товары возвращаются на склад.
- При доставке заказа автоматически создается запись в таблице `Доставки`.
- При завершении доставки заказ переводится в финальный статус.

## Источники данных

Начальные данные хранятся в `data/*.csv`. Структура базы хранится в `sql/schema.sql`, а именованные запросы для таблиц и отчетов - в `sql/queries.sql`.
